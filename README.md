# Positive Vibration - Mental Health Services Website

A premium, modern, and clean multipage website designed for "Positive Vibration," a mental health and wellness center.

## ✨ Features

- **HSL-Based Theme**: Change the entire site's color scheme in one place.
- **Responsive Layout**: Optimized for mobile, tablet, and desktop environments.
- **Modern Design**: Glassmorphism, smooth gradients, and elegant typography (Outfit & Inter).
- **Interactive**: Scroll-triggered animations and sticky navigation.

## 🚀 Getting Started

To run the project locally, ensure you have [Node.js](https://nodejs.org/) installed, and then:

1. Clone this repository:
   ```bash
   git clone https://github.com/darthdragneel1/Posoitive-Vibration-Website.git
   ```
2. Navigate to the project folder and start the dev server:
   ```bash
   npm run dev
   ```

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **Vanilla CSS**: Custom-designed system and HSL variables.
- **JavaScript**: Interactive elements and reveal animations.
- **AI-Generated Imagery**: Serene, high-quality visuals specifically designed for this site.
